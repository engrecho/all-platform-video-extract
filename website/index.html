特性

为什么选择它

从解析到下载，一个工具搞定全部流程。不装依赖，不解压运行库。

零依赖运行

仅依赖 Node.js 内置模块（crypto / fetch / child_process），无需 npm install，下载即用。

1000+ 平台

抖音、快手、B站、YouTube、TikTok、小红书、微博、公众号等主流平台全覆盖，未列出的也能自动识别。

一键下载

解析后自动下载视频、音频、封面、所有图片到独立目录，公众号文章自动生成 markdown 离线副本。

加密发布

发布版脚本经 AES-256-CBC 加密，源码不可直接查看。开发版与发布版分离，一键构建打包。
